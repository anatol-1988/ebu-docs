ЭБУ ГМП
*******


Описание
========

ЭБУ ГМП ЛБИЕ.453639.004 является перепрограммируемым электронным устройством
управления на основе процессора SC186EX с тактовой частотой 96 МГц, рабочей
памятью 8 Мб, памятью программ 8 Мб.
